Was ist Swing?
Swing ist ein vorgefertigtes Paket, um mit Java graphisch funktionierende Programme zu schreiben.
Swing stellt eine große Menge von Klassen, Interfaces und Methoden zur Verfügung mit denen.



Befehle:

Mit der Klasse JFrame kann man graphische Fenster mit Java erstellen:

< JFrame fenster = new JFrame("Titel"); >

Man erzeugt also ein Objekt der Klasse JFrame und weißt der Variablen fenster die Referenz auf dieses Objekt zu.
Damit das Fenster auch wirklich angezeigt wird, muss der Bool'sche Wert <setVisible> auf <true< gesetzt werden:
< fenster.setVisible(true) >

Um das Fenster in Bereiche einzuteilen, wird die Klasse JPanel benutzt:
<JPanel leftPanel = new JPanel();>


Ein graphisches Objekt wird dann mit dem Befehl <fenster.add(objektreferenz)> oder 
<leftPanel.add(Objektreferenz) in das Fenster bzw den Bereich eingebaut.

Dafür gibt es verschiedene Möglichkeiten:
Eine Graphik (*.jpg, .*png, ...) kann eingefügt werden, wenn man ein neues Objekt der Klasse <BufferedImage> erzeugt:
<BufferedImage bild = new BufferedImage(new File("name.jpg")); > ???

Um gewöhnlichen Text auch in graphischen Fenstern anzeigen zu können, gibt es den Befehl JLabel:
<JLabel text = new JLabel("Hier steht jetzt ein Text"); >

Außerdem kann man Schalflächen nutzen. Dazu müssen allerdings einige Anweisungen gegeben werden:
<
JButton buttonName = new JButton("Hier klicken, um das Programm zu beenden");
ActionListener terminateListener = new ActionListener(){
 public void actionPerformed(ActionEvent e){
	System.exit(0); //beendet das Programm
	}
}
buttonName.addActionListener(terminateListener);
>
Zur Erklärung: Man muss einen Listener (="Horcher") einrichten, der bekannt wenn ein bestimmtes Ereignis eintritt.
ActionListeners werden benutzt, um zu erfahren, wenn auf eine Schaltfläche geklickt wurde. 
In der darauf folgenden Methode actionPerformed wrid angegeben, was passieren soll, wenn auf den Button geklickt wird.
Zuletzt muss man den Listener dem Button "vorstellen".
Damit der Button angezeigt wird (genauso wie die Bilder und Labels), muss das Objekt wie gesagt noch dem JFrame
hinzugefügt werden:
<fenster.add(buttonName);>

Ebenfalls kann mit einem Listener Bewegungen und Aktionen der Maus abfragen und verwenden.
